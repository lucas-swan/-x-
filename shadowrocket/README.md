## 小火箭说明配置
### 设置
1. 延迟测试方法

2. 订阅
      - 打开时更新--打开
      - 自动后台更新--打开
      - 根据Ping排序--打开
  
### 配置
1. 右上角＋号
2. 填入地址点击下载
3. 远程文件处出现新增的配置文件--点击改文件--使用配置（若出现“未完成该操作。无效的自变量”，打开VPN后在重新操作此步即可）
